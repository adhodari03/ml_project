THE ATTENTION MECHANISM
Attention(Q, K, V) = softmax(QKᵀ / √dₖ) · V
Q (Query), K (Key), V (Value) are linear projections of the input.

Why scale by √dₖ? Without scaling, dot products grow proportionally to dₖ (dimension of keys). Large dot products push softmax into saturated regions where gradients vanish, making training unstable. Dividing by √dₖ keeps variance ≈ 1.

Multi-Head: 32 heads run in parallel, each with dₖ=128. Different heads learn different relationships — some track syntax, others semantics, others positional patterns.

GQA: Llama 3 uses Grouped-Query Attention — 32 query heads share 8 KV heads, reducing memory by 4× during inference while maintaining quality.
